À quoi sert le HTML ?

Le HTML sert à faire un squelette du site, comme l'image ci-dessous :

Donc, le header est la tête de page, le footer est le bas de page, la balise "nav" est pour y mettre des liens principaux (pas obligatoire), la section est une balise à peu près égale à la balise "div", la balise "aside" est utilisée pour rajouté des informations complémentaires et peut aussi jouer le rôle d’une colonne latéral (sidebar) dans un article.
