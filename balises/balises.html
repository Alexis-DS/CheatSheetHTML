Les balises HTML utiles

Les balises

<!-- --> : Commentaire.

<!DOCTYPE html> : Type de document: html

<html> : Obligatoire au debut et la fin du fichier HTML.

<head> : En-tête d'un fichier HTML (encadre les balises <title>, <meta>, ...).

<meta> : Information sur la page.

<meta charset="utf-8"> : Voir les caractères spéciaux et les lettres avec accents.

<title> : Titre du document HTML (à insérer dans la balise <head>)

<body> : Encadre le corps d'un fichier HTML.

<h1> : Titre de niveau 1.

<h2> : Titre de niveau 2.

<h3> : Titre de niveau 3.

<h4> : Titre de niveau 4.

<h5> : Titre de niveau 5.

<h6> : Titre de niveau 6.

<p> : Début d'un paragraphe, saute 2 lignes.

<a> : Création d'une ancre pour liens hypertexte.

<table> : Faire un tableau.

<tr> : Nouvelle ligne d'un tableau (encadré par la balise <table>).

<td> : Nouvelle cellule dans un tableau (encadré par la balise <tr>)

<strong> : Mettre le texte ou une partie du texte en gras.

<span> : Implémente une feuille de style.

<footer> : Pied de page d'un document HTML.
